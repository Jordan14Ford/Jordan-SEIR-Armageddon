# 2A — Raw CLI proof

This folder is for **raw CLI proof** (terminal output) that does not fit `curl_checks/` or `aws_checks/`.

- **curl/dig** → use [curl_checks/](../curl_checks/)
- **AWS CLI** → use [aws_checks/](../aws_checks/)
- Other CLI (e.g. terraform output, custom scripts) → place here.

If empty, no additional CLI proof was required beyond curl and AWS checks.
